# 99乘法表 - 两种语句实现
这是一个用c语言编写的99乘法表程序（for语句版和while语句版）
##版本列表
### 1.for循环版本(`multiplication.c`)
使用for循环实现的版本
### 2.while循环版本
使用while循环实现的版本
## 功能特点
- 打印完整的99乘法表
- 对齐美观
- 代码简洁易懂
## 编译和运行
### for循环版本
```bash
# 编译
gcc -o multiplication_for multiplication.c
# 运行
./multiplication_for
### while循环版本
```bash
# 编译
gcc -o multiplication_while multiplication_while.c
# 运行
./multiplication_while
